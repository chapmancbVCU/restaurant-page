formatPhoneNumber(value)

Function that receives number as an input and formats it based on the length. For strings less than three it does nothing. For values less than 7 you get a formatted string that looks like xxx-xxx. If the input string is more than 7 then the formatted string looks like xxx-xxx-xx. Client side validation is recommened as well. Do not call this function directly within the html file. The phoneNumberFormatter function will be used to call this function in order to set the correct format.
Parameters:
Name Type Description
value string the string of numbers
